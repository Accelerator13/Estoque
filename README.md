O **Sistema de Estoque** é uma aplicação voltada para o gerenciamento inteligente de mercadorias, controle de entradas e saídas de produtos e otimização operacional para estabelecimentos comerciais. O projeto foi estruturado com foco em performance, facilidade de uso e organização de dados.

---

## 🚀 Funcionalidades Principais

- **📦 Cadastro de Produtos:** Inclusão, edição e exclusão de itens com especificações detalhadas (nome, preço, quantidade, categoria).
- **🔄 Movimentação de Estoque:** Registro em tempo real de entradas (compras/reposição) e saídas (vendas/baixas).
- **⚠️ Alertas Automáticos:** Notificações visuais quando um produto atinge a quantidade mínima configurada no estoque.
- **📊 Relatórios e Consultas:** Visualização simplificada da saúde atual do inventário e histórico recente de movimentações.
